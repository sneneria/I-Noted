Features

Signup & Login

The user is required to register first in order to access the application. Users will input their username, email address, and password. If the user has already registered to the application, there's an option below that lets them log in directly. Upon the user has successfully registered to the app, the user will be prompted to the login page. Legitimate login credentials will lead the users to access the application. If the users haven't registered yet, there's an option below that let the user register to the application.

Menu/Tags

The menu of the application, here the user can browse subjects/tags and can select what notes they want and they can press the back button if they are not certain of what they are looking for.

Browse Tags 2

The user selects a specific subject or tag, the tag that the user selects will have the notes that belong to it.

Browse Tags 3

The user has selected a certain note that can be freely used. It is also you can next the notes when the user is already done.

Share Notes

the user can share notes to a specified tag or subject. And the user can upload the file with a pdf or any other tools.
